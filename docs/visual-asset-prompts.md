# Visual Asset Text-to-Image Prompt Pack (v1)

Use these prompts to generate source images for the full required asset pack in `/Users/jacob/Projects/planted-tank-lab/docs/visual-asset-pack-spec.md`.

## Generation Defaults (apply to every prompt)

- Style: photoreal aquascaping reference, neutral studio lighting, high detail.
- Composition: single isolated subject, centered, side/front aquarium viewing angle.
- Background: flat clean light-gray background for easy Canva cutout.
- Framing: subject fills about 75-85% of frame, nothing clipped.
- Output quality: sharp edges, no motion blur, no text/logos/watermarks.

Use this negative prompt for all renders:

`text, watermark, logo, frame border, multiple objects if not requested, people, fish, reflections of camera, heavy cast shadows, low resolution, blurry, cartoon, illustration`

## Quick Generation Workflow

1. Generate every PNG using prompts above.
2. In Canva, remove background cleanly and export transparent PNG for hardscape/plants.
3. Resize/crop to the exact spec:
   - hardscape: `2048x2048`
   - plants: `1536x2048`
   - substrate tiles: `2048x2048`
4. Save using exact paths/filenames listed in this file.
5. Create folder-level `manifest.json` files per the schema in `/Users/jacob/Projects/planted-tank-lab/docs/visual-asset-pack-spec.md`.
